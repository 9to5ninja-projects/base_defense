# Skyguard: Cell Defense

**Skyguard: Cell Defense** is a strategic grid-based tower defense game where you must build and manage a vertical city to defend against relentless waves of aerial and ground enemies.

## Features

*   **Vertical City Building**: Construct a multi-layered defense grid. Stack buildings to maximize space and efficiency.
*   **Energy Management**: Balance your power grid. Production buildings fuel your defenses and shields. Negative energy drains your credits!
*   **Shield System**: Maintain a powerful energy shield to protect your city from bombardment.
*   **Combined Arms Defense**:
    *   **Turrets**: Anti-air defenses with upgradeable tiers.
    *   **Drones**: Mobile interceptors that hunt down aerial threats.
    *   **Barracks**: Spawn ground defenders to hold the line against invaders.
*   **Ground Combat**: Defend against Kamikaze enemies that crash land and spawn ground invaders.
*   **Progression**: Unlock new columns, upgrade buildings, and manage your economy to survive increasingly difficult waves.
*   **Boss Battles**: Face off against massive Boss enemies every 10 waves.

## Controls

### General
*   **Arrow Keys**: Move Cursor / Navigate Menus
*   **Space / Enter**: Confirm / Open Menu
*   **Esc**: Cancel / Pause / Back

### Build Mode
*   **Space**: Open Build Menu (on empty cell) / Open Context Menu (on building)
*   **1-6**: Quick Build Hotkeys
    *   [1] Power Plant
    *   [2] Datacenter
    *   [3] Capacitor
    *   [4] Turret
    *   [5] Drone Factory
    *   [6] Barracks
*   **M**: Move Building
*   **U**: Upgrade Building
*   **R**: Repair Building
*   **Del / Backspace**: Sell Building
*   **W**: Start Next Wave
*   **H**: Toggle Field Manual

## Installation & Running

1.  Ensure you have Python 3.x installed.
2.  Install dependencies:
    ```bash
    pip install pygame
    ```
3.  Run the game:
    ```bash
    python run.py
    ```
